LOCAL -->  cat airports.csv | python mapper.py | sort -k1,1 | python reducer.py

HDFS  -->  hadoop jar /usr/lib/hadoop-0.20-mapreduce/contrib/streaming/hadoop-streaming-2.0.0-mr1-cdh4.1.1.jar -file "path to mapper file" "path to reducer file" -mapper "python mapper.py" -reduce "python reducer.py -input "path to input file" -output "path to output directory"


P.S. - Output directory should not be created from earlier or else it will throw error.
